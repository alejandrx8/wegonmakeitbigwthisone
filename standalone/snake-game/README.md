# Snake

A classic Snake mini-game built with React + Vite + TypeScript + Tailwind CSS v4.

## Features

- 20×20 grid, growing snake, food spawning
- 10 levels — snake speeds up every 5 foods eaten
- Score system (level × 10 per food)
- High score saved to localStorage
- Floating "+score" indicators
- Keyboard controls (Arrow keys / WASD, Space to pause, Enter to start)
- Mobile D-pad for touch devices
- Dark neon theme

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Controls

- **Arrow keys / WASD** — move the snake
- **Space / Esc** — pause / resume
- **Enter** — start / restart
